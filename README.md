Q. What is the difference between var, let, and const?
=>
var : It is old way to declare variables. It can be redeclare and change value. It is not a block scoped. It is a global scope
let : It is the modern way to declare variables. It cannot be redeclare, but can change value. It is a block scoped. It is more safe than var.
const : It is used for fixed values. It cannot be redeclare or change value. It is block scoped.

Q. What is the spread operator (...)?
=>
The spread operator (...) is used to expand or unpack values from an array or object.

Q. What is the difference between map(), filter(), and forEach()?
=>
map() : It creates a new array by transforming each element. It returns new array. It is used when want to change data.
filter() : It creates a new array with only matching elements. It returns new array. It is used to select specific data.
forEach() : It is just loops through array. It does not return anything. It is used for side effects like update, UI etc.

Q. What is an arrow function?
=>
Arrow function is a short way to write a function in JavaScript using =>. It helps to write less code and cleaner code.

Q. What are template literals?
=>
Template literals are a way to write strings easily in JavaScript using backticks `, and can insert variables inside.
